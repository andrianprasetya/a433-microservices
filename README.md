# karsajobs-ui
Frontend for [karsajobs](https://github.com/dicodingacademy/karsajobs)

## Project setup
```
npm install
```

### 1. Compiles and hot-reloads for development
```
npm run serve
```

### 2. Compiles and minifies for production
```
npm run build
```

### 3. Lints and fixes files
```
npm run lint
```
